A keybinding tool for defending quickly in NationStates.


Quick Start Full Introduction

What is Gauntlet?

Gauntlet is a browser extension that allows you to set custom keybinds for common actions in defending gameplay. For instance, with Gauntlet, you can prep your switchers, move between regions, and refresh your reports with just your keyboard.

How do I get started?

Gauntlet comes with default keybinds, which you can view in your Gauntlet settings. Although these defaults should feel familiar if you’ve ever used Breeze++, you can always customize them to your liking.

In addition to keybind settings, some Gauntlet keybinds need a jump point or a list of switchers to work properly. You can customize both of these settings as well in your Gauntlet settings.

What if I have more questions?

If you’d like to return to this page, you can bookmark it. However, the Gauntlet settings contain the full list of keybinds and settings and short descriptions of their functionality as necessary.

If you have any bug reports or feature requests that you’d like to share, feel free to file an issue on GitHub. If you’re not familiar with GitHub, it’s a website for hosting source code, including the source code for Gauntlet. You can create a GitHub account for free.
